# Beginner ML Projects

Just starting out with machine learning. These are my first two projects as I learn the basics.

## Projects

### 1. Iris Flower Classifier
A simple classification model using a Decision Tree to predict iris flower species.
- Dataset: sklearn's built-in Iris dataset
- Algorithm: Decision Tree Classifier
- Run: `python iris-classifier/iris_classifier.py`

### 2. House Price Predictor
A regression model using Linear Regression to predict house prices from features like size and number of rooms.
- Dataset: Synthetic (generated with numpy)
- Algorithm: Linear Regression
- Run: `python house-price-predictor/house_price_predictor.py`
